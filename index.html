    How backups work

    1. Each backup writes a timestamped file into a backups/ folder, plus updates backup.json as the latest pointer.
    2. Auto-restore on open loads the latest on startup so a second device stays current.
    3. Version History lists every saved version. Restore any earlier one if needed.
    4. The app keeps your most recent 30 versions and prunes older ones automatically.
